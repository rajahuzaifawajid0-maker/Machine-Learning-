The process

Eight steps below the waterline

Every stage the raw CSV passes through before it's fit to train a model — in the order the pipeline actually runs.

01
Ingest

Load and inspect the manifest

Read the raw CSV into a dataframe and take a first look at its shape and columns.

891 rows × 12 columns
02
Audit

Find the gaps in the record

Count nulls per column — and catch a stray value hiding where a port code should be.

Age 177 · Cabin 687 · Embarked 2 missing, 1 malformed
03
Repair

Fill the missing entries

Age gaps take the column mean, empty cabins become "Unknown", and any port that isn't S, C, or Q is treated as missing and set to the most common port.

Age → mean (29.7) · Cabin → "Unknown" · Embarked → mode
04
Encode

Turn text into numbers

Sex and Embarked are label-encoded so every field a model reads is numeric.

LabelEncoder · Sex, Embarked
05
Cast

Tidy the data types

Age and Fare are rounded down to whole numbers, trimming decimal noise.

Age, Fare → int
06
Scale

Bring every column to one range

Seven numeric columns are rescaled so no single feature dominates on magnitude alone.

MinMaxScaler · range −1 to 1
07
Split

Set aside a test set

The cleaned table is divided into training and holdout sets before any model sees it.

80 / 20 split · random_state 42
08
Export

Write the finished table

The cleaned, encoded, scaled dataframe is saved to disk, ready for modeling.
